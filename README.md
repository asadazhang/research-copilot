# research-copilot

An AI-powered Research Workspace for Social Science Students.

## Vision

Research Copilot aims to streamline the academic research workflow by integrating literature reading, note-taking, citation management, and AI-assisted writing into a single platform.

## Features

- PDF Reader
- AI Summary
- Reading Journal
- Paper Chat
- Citation Generator
- Literature Matrix
- Knowledge Graph

## Tech Stack

Frontend:
- React
- Next.js

Backend:
- Python
- FastAPI

Database:
- PostgreSQL

AI:
- LLM API
- Vector Database
